# Bit Juggling
Numeral system converter and bitwise calculator for Windows. List of operations: bitwise **NOT** (*complement*), bitwise **OR** (*logical inclusive OR*), bitwise **AND** (*logical AND*), bitwise **XOR** (*logical exclusive OR*).
